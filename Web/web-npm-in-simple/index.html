6、缓存管理

当指定 npm install 安装一个第三方包的时候，npm 会在本地缓存目录保存这个包的一个拷贝，这样，在下次我们执行安装的时候，如果本地缓存已经有这个包，就无需从网络获取了。
这个缓存目录通常是 ~/.npm，所有我们安装过的第三方包都会在这里保存一个拷贝。
